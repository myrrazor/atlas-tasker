## D-1734 — Classification Label Filenames Are Hash-Suffixed

- Date: 2026-05-06
- Question: Should classification label filenames be human slugs only?
- Options Considered:
  - Use sanitized `kind:id` slugs for readable filenames.
  - Use readable slugs plus a short hash over the exact entity kind and ID.
- Chosen Option: PR-705 uses `class-<slug>-<hash>.md`, where the hash is computed from the exact entity kind and ID.
