# v1.8.1 Release Owner Closeout

This closeout is intentionally not a feature train. It records the remaining owner actions required before Atlas Tasker can be announced as a public release candidate.

Current decision: **no public ship**.

## Hard Blockers

### 1. Hosted Release Proof

Current status: blocked.

Latest checks from this environment:

```text
VERSION=v1.8.0-rc1 sh scripts/preflight-release.sh --hosted
preflight_status=blocked
reason_code=workflow_permissions_unreadable
```

```text
gh release view v1.8.0-rc1 -R myrrazor/atlas-tasker --json tagName,isPrerelease,isDraft,assets,url
release not found
```

Release owner actions:

- fix GitHub workflow/release permissions
- cut the real `v1.8.0-rc1` prerelease tag
- wait for GitHub Actions to publish archives and `checksums.txt`
- verify hosted checksums
- verify GitHub artifact attestations
- install from hosted assets
- run packaged smoke from the installed hosted binary
- record hosted proof in `docs/release/v1.8-release-evidence.md`

### 2. License Selection

Current status: blocked.

There is no committed `LICENSE` file. Until the owner chooses and commits one, the project is visible source, not clearly open source.

Owner decision required:

- MIT: simplest permissive path and broad adoption.
- Apache-2.0: permissive path with explicit patent language.
- GPL/AGPL: copyleft path only if that obligation is intentional.

No agent should choose this automatically. After owner selection, commit the exact license text as `LICENSE` and update README project status.

### 3. Human Documentation And Aesthetic Review

Current status: blocked pending owner review.

Required human pass:

- rendered GitHub README
- rendered docs landing and release docs
- quickstart flow from a clean clone
- demo transcript and prompt packs
- `tracker board --md`
- `tracker ticket view --md`
- `tracker goal brief APP-1 --md`
- `tracker goal manifest APP-1 --actor human:owner --reason "review goal" --md`
- `tracker tui`

Codex can verify commands, snippets, links, and leakage. A human still needs to decide whether the first impression feels excellent.
