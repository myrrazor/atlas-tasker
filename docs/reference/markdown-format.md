# Markdown Format

Markdown output is for humans and coding agents. It should be readable when pasted into Codex, Claude Code, GitHub comments, or a handoff document.

Markdown output is not canonical storage. The canonical state lives in Atlas workspace files and event records.

Current Markdown surfaces include goal briefs/manifests, selected reports, handoffs, gates, and release docs. Sensitive tickets should be redacted before sharing Markdown; goal output is not a redaction boundary.
